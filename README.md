# tableworld

<img src="tw.jpg" width="600">

A little [MUD](https://en.wikipedia.org/wiki/Multi-user_dungeon)- and
[MUDDL](https://if50.substack.com/p/1980-mud)-inspired game.

# building it

Install Java, Leiningen, make.  Then,

    make

# running it

    ./tw

# playing it

    telnet 127.0.0.1 9999
